### Task 6: Render Timing - ✅ IMPLEMENTED

**Location**: `crates/tui/src/lib.rs:446-457`

```rust
#[cfg(debug_assertions)]
fn log_render_timing(&self, elapsed: std::time::Duration) {
    if self.config.debug.render_timing {
        let ms = elapsed.as_millis() as u64;
        if ms > 16 {
            eprintln!("WARN: Render took {}ms (target <16ms)", ms);
        }
    }
}
```

**Config**: `crates/config/src/lib.rs:1361` - `debug.render_timing: bool`
