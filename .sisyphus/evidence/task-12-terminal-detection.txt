# Task 12: Terminal Background Detection - Evidence

## Implementation Summary

Added `terminal_color_scheme()` function in `crates/app/src/style/theme.rs` that:
1. Detects terminal color scheme from `$COLORFGBG` environment variable
2. Parses "foreground;background" format (e.g., "0;15" = dark, "15;0" = light)
3. Uses caching via `OnceLock` to ensure detection happens once at startup
4. Defaults to `ColorScheme::Dark` if detection fails

## Key Types Added

```rust
pub enum ColorScheme {
    Dark,
    Light,
}
```

## Key Functions Added

```rust
fn detect_terminal_color_scheme() -> ColorScheme  // private, does actual detection
pub fn terminal_color_scheme() -> ColorScheme     // public, returns cached result
```

## Detection Logic

- COLORFGBG format: "foreground;background" 
- If background > 7 → Dark theme
- If background <= 7 → Light theme
- Default: Dark
